<h1>Contact`s book</h1>
<hr>
by Rudniev Roman
<hr>
This project is designed to allow you to add and
edit your contacts in a table format.
<hr>
<h2>The table has the following sections:</h2>
<ul>
     <li>№</li>
     <li>Photo</li>
     <li>First name</li>
     <li>Last name</li>
     <li>Phone number</li>
     <li>Email</li>
     <li>Nationality</li>
</ul>
